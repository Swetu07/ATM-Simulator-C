# ATM Simulator in C

A console-based ATM Simulator developed in C using structures, functions, and file handling.

## Features

- Create Account
- Login Authentication
- Check Balance
- Deposit Money
- Withdraw Money
- Logout
- Persistent Data Storage using File Handling

## Technologies Used

- C Programming
- Structures
- Functions
- File Handling

## Project Structure

```
ATM-Simulator-C/
├── main.c
├── account.c
├── account.h
├── README.md
└── .gitignore
```

## Learning Outcomes

- Menu-driven programming
- Structures
- Functions
- File handling
- Authentication
- Modular programming

## Compile

```bash
gcc main.c account.c -o ATM-Simulator
```

## Run

```bash
ATM-Simulator.exe
```
